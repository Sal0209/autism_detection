Autism Detection using Machine Learning
📌 Overview

This project focuses on predicting Autism Spectrum Disorder (ASD) traits using a questionnaire-based machine learning approach. The goal is to support early screening and assist healthcare professionals in identifying potential autism indicators.

🚀 Features

Questionnaire-based autism prediction

Data preprocessing and feature engineering

Machine learning model for classification

Real-time prediction using user input


🛠️ Tech Stack

Python

Pandas, NumPy, Scikit-learn



📊 Model

Multiple algorithms were tested, and Random Forest was selected due to its high accuracy and reliability.


Integration with healthcare systems
